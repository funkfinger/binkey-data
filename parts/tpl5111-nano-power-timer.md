## Details

The Texas Instruments TPL5111 is a nano-power system timer designed for power gating applications in ultra-low-power battery-powered devices. With an exceptionally low quiescent current of just 35nA, it's ideal for IoT devices, wireless sensor nodes, and energy-harvesting applications that require periodic wake-up and power management.

## Description

The TPL5111 is a programmable timer IC that enables efficient power management by periodically enabling and disabling power to a system. It features an internal oscillator and can be configured with external resistors to set custom delay periods. The device is designed to minimize power consumption during standby, making it perfect for battery-powered applications that need to operate for extended periods.

## Specifications

- **Type**: Nano-Power System Timer for Power Gating
- **Package**: SOT-23-6 (6-pin surface mount)
- **Supply Voltage**: 2.0V to 5.5V (typical 2.5V)
- **Quiescent Current**: 35nA (ultra-low power)
- **Output**: Reset/Enable output for controlling system power
- **Programmable Delay**: Configurable via external resistor

## Pin Configuration (SOT-23-6)

- **Pin 1**: VDD (Power Supply)
- **Pin 2**: GND (Ground)
- **Pin 3**: DELAY (Timing resistor input)
- **Pin 4**: RESET (Output - active low)
- **Pin 5**: ENABLE (Input - active high)
- **Pin 6**: NC (No Connect)
