## Description

The Same Sky PTN091 is a standard 9.5mm vertical rotary potentiometer with a 10K ohm linear taper. It features a 15mm knurled shaft for comfortable manual adjustment and is designed for through-hole PCB mounting. This is a reliable, general-purpose potentiometer suitable for volume controls, sensor calibration, parameter adjustment, and other applications requiring variable resistance.
