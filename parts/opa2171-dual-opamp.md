## Description

The Texas Instruments OPA2171 is a dual general-purpose operational amplifier designed for single-supply operation with a wide supply voltage range of 2.7V to 36V. This low-power op-amp features rail-to-rail output capability and is ideal for battery-powered applications, sensor signal conditioning, and general analog circuit design. The VSSOP-8 surface-mount package provides a compact solution for space-constrained designs.

## Circuit Integration

### Basic Non-Inverting Amplifier

Connect for single-supply operation:
- V+ to positive supply (2.7V to 36V)
- GND to ground
- Input signal to non-inverting input
- Feedback network for desired gain

### Single-Supply Biasing

For AC-coupled single-supply circuits:
- Bias non-inverting input to mid-supply voltage
- Use coupling capacitors on inputs/outputs
- Allows full dynamic range utilization
